تطبيق لسطح المكتب يعمل على نظام تشغيل **Windows**، يهدف إلى تذكير المسلم بالأذكار والأدعية اليومية بشكل جمالي وهادئ دون مقاطعة العمل.

تم تصميم هذا التطبيق ليعرض الأذكار بأسلوب حديث مستوحى من واجهات المستخدم الفاخرة (Premium Glassmorphism).

---

## ✨ المميزات الرئيسية
* **خلفية بلور تفاعلية (Dynamic Blurred Background):** يقوم التطبيق بأخذ لقطة من شاشتك الحالية وتطبيق تأثير البلور الضبابي (Blur Effect) التلقائي، لتبدو الواجهة مدمجة بشكل ساحر مع ما تعمل عليه دون تشتيت.
* **مظاهر متغيرة حسب الوقت (Time-based Themes):** تتغير درجات ألوان الواجهة والإضاءة المحيطة تلقائياً بناءً على وقت اليوم (الصباح، الظهر، المساء، والليل).
* **تأثير التنفس المضيء (Breathing Glow Effect):** نبضات ضوئية خفيفة تتحكم بوهج النص والظل الخلفي لتعطي شعوراً حياً وتفاعلياً فخماً.
* **تحكم مرن بالماوس ولوحة المفاتيح:** يمكنك إغلاق التذكير فوراً بمجرد الضغط في أي مكان بالماوس، أو بالضغط على زر `Esc` أو `Space` في لوحة المفاتيح.
* **نافذة إعدادات متكاملة:**
  * تفعيل أو إيقاف التنبيهات مؤقتاً.
  * تعديل الفاصل الزمني لظهور الأذكار (من 1 إلى 240 دقيقة).
  * تعديل مدة بقاء الذكر على الشاشة (من 3 إلى 90 ثانية).
  * التحكم في قوة تأثير البلور (Blur Radius) ودرجة تعتيم الخلفية.
  * التحكم بنعومة وسرعة حركة الدخول والتلاشي (Fade & Slide duration).
  * العرض على شاشة واحدة أو على جميع الشاشات المتصلة بالكمبيوتر (Multi-screen support).
* **التشغيل التلقائي مع ويندوز (Startup Support):** خيار مدمج لتشغيل التطبيق تلقائياً مع إقلاع النظام عبر اختصارات آمنة.
* **أيقونة شريط المهام (System Tray):** يعمل التطبيق بهدوء في الخلفية بجانب الساعة مع قائمة سريعة لإظهار ذكر فوراً، فتح الإعدادات، أو إيقاف التشغيل.

## 🚀 التثبيت والتشغيل السريع

### 📥 الطريقة الأولى: استخدام مثبت ويندوز الجاهز (الموصى بها)
إذا كنت مستخدماً عادياً، يمكنك تحميل برنامج التثبيت الجاهز مباشرة:
1. اذهب إلى صفحة **[Releases]** في هذا المستودع.
2. قم بتحميل الملف **`DhikrMomentSetup.exe`**.
3. قم بتشغيل الملف واتبع خطوات التثبيت البسيطة (يدعم إنشاء اختصار على سطح المكتب وتفعيله مع إقلاع الويندوز).

### 🛠️ الطريقة الثانية: التشغيل من خلال كود المصدر (للمطورين)
إذا كنت مطوراً وترغب في تعديل الكود أو تشغيله مباشرة:

#### المتطلبات الأساسية:
* مثبت بايثون الإصدار Python 3.10 أو أحدث.
* مكتبة PyQt6.

#### خطوات التشغيل:
1. قم بإنشاء بيئة وهمية وتفعيلها:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. تثبيت المكتبات المطلوبة:
   ```bash
   pip install PyQt6
   ```
3. تشغيل التطبيق:
   ```bash
   python لحظة_ذكر.py
   ```

## 📦 بناء ملف التثبيت الخاص بك
المشروع مجهز بسكريبت بايثون لبناء الملف التنفيذي ومثبت Inno Setup تلقائياً:
1. تأكد من تثبيت برنامج **[Inno Setup 6](https://jrsoftware.org/isinfo.php)** على جهازك.
2. قم بتثبيت مكتبة `PyInstaller`:
   ```bash
   pip install pyinstaller
   ```
3. قم بتشغيل سكريبت البناء:
   ```bash
   python build_dhikr_moment_installer.py
   ```
سيقوم السكريبت بتجميع التطبيق باستخدام PyInstaller ثم استدعاء Inno Setup لإنتاج ملف `DhikrMomentSetup.exe` داخل مجلد `release/`.
